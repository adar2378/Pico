Introducing PICO.

A stupidly simple & blazing fast, flat file CMS.

Making the web easy.

Stupidly Simple

Picos makes creating and maintaining a website as simple as editing text files.

Blazing Fast

Pico is seriously lightweight and doesn't use a database, making it super fast.

No Database

Pico is "flat file" CMS, meaning no database woe's no MySQL queries, nothing.

Markdown Formatting

Edit your website in your favorite editor using simple Markdown formatting.

Twig Templates

Pico uses the Twig templating engine, for powerful and flexible themese.

Open Source

Pico is completely free and open source, released under the MIT license.

Want to contribute?

Help make Pico better by checking out the GitHub Repository and submitting pull requests.
